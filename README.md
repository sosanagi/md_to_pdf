# md_to_pdf

## 設定

- 拡張機能からmarkdown-pdfをインポート
- mdファイルをpdf変換する

## 出力方法
- mdファイルを開く

- 「cmd + Shift + p」コマンドパレットを開く

```
> Markdown PDF: Export (settings.json)
```
を選択する
